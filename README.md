# Yield Curve Forecasting

## Project Overview

This project forecasts U.S. Treasury yield curves using statistical, machine learning, and deep learning models.

The project analyzes:
- 2-Year Treasury Yield
- 10-Year Treasury Yield
- Yield Spread

It also compares forecasting performance using:
- ARIMA
- XGBoost
- LSTM

A Streamlit dashboard is included for visualization and model comparison.
